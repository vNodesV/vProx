Helper — Chain

Writes one chain profile file: config/vlog/chains/<chain_name>.toml. Required identity fields are chain_name and chain_id, plus at least one [[chain_services]] entry.

[[chain_services]] Managed Services

Each service row requires name and service_type. validator rows require valoper, and internal_ip is required when link_to_vm is enabled.

Tip
Chain and infra files may be multiple. This form writes one chain profile file under config/vlog/chains based on chain_name.

Helper — Infra

Writes config/infra/<datacenter>.toml. [host] is optional; each VM row should include at least name and host.

[host] Physical Server (optional)
[vprox] vProx Instance
[[vm]] Virtual Machines

Each VM row requires name and host. Empty rows are ignored.

Tip
This step writes config/infra/<datacenter>.toml using [host], [vprox], and [[vm]] entries.
